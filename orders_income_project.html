    • Project goal: Partial automation of order and income data updates (for analysis and calculations) to reduce manual labor and errors in small business accounting.

    * I can't attach order, profit, and accounting files because they contain commercial and customer data. Later I'll add these files with fake data to demonstrate how the script works.

    Description: TikTok Shop is an e-commerce platform (similar to Amazon). From the profile, sellers can download Excel files containing orders in their store and payments for them. My friend, a current seller, asked me to work with his accounting department. This particular project allows a seller to receive data from order and payment files, process them properly, and upload them to the accounting file for analysis and calculations.

    Advantages:
    1. The script selects only the necessary columns from the files (the original files contain a information that is unnecessary for calculations)
    2. It allows a seller to update the data in the accounting file, removing duplicates and saving only fresh data, since information on payments can change over time depending on deductions, returns, etc.
    3. It sets the necessary data types for specific columns, which prevents duplicates (due to scientific notation in the columns with the order number and SKU) and avoids damaging calculations in Excel formulas on other sheets during the conversion of numeric columns to text.

    Possible further improvements:
    1. Using the API, eliminate the steps for downloading order and income files and receive data directly.
    2. As the business grows and the number of orders increases, working with Excel can become quite slow, so using the same principles, I can adapt the script to write data to the database.

    Conclusion:
    It is a small project, but at the same time, it is not an educational project; it is a specific task that solves real business problems, eliminates manual data entry, and frees up a huge amount of time for employees.
